#### Install + Deploy

1. Install and load the `devtools` package in `R`, which will facilitate the installation of this package.
2. Install **hetviz**.
3. Load **hetviz**.
4. Run **hetviz**.
```r
# step 1
install.packages("devtools")
# step 2
devtools::install_github("sanoke/hetviz")
# step 3
library(hetviz)
# step 4
hetviz()
```
